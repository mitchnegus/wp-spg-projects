A plugin to showcase the projects (past and present) of the Science Policy Group at Berkeley.


== Description ==

This is a plugin for managing projects on the Science Policy Group at Berkeley Wordpress site.
It allows a site administrator to add project information (including a description, contact, logo, and time-period).
This project information is then displayed on modified custom post archive page.

To see the plugin in action, check out the [Science Policy Group at Berkeley](https://sciencepolicy.berkeley.edu/projects) website!

== Installation ==

At the moment, this plugin is not available through WordPress.org.
Instead, you must clone or download the repository manually. Those steps are

1. Clone/upload this repository (with file `projects.php` to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
